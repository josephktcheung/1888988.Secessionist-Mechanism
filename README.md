# Secessionist Mechanism Bug Report

# 分离主义机制错误报告

This folder contains the bug report, improvement suggestions, and technical implementation for the Secessionist mechanism issues in EU5.

本文件夹包含 EU5 分离主义机制问题的错误报告、改进建议和技术实现。

## File Structure / 文件结构

### Core Documents / 核心文档

- **`report.md`** - Source bug report with localization keys
- **`report_en.md`** - English translation
- **`report_zh.md`** - Chinese translation
- **`report_source.md`** - Original source (backup)

- **`suggestion.md`** - Source improvement suggestion with localization keys
- **`suggestion_en.md`** - English translation
- **`suggestion_zh.md`** - Chinese translation

- **`technical implementation.md`** - Source technical implementation
- **`technical implementation_en.md`** - English translation
- **`technical implementation_zh.md`** - Chinese translation

### Subfolders / 子文件夹

- **`archive/`** - Old/backup files (`.txt` files, old translations)
- **`bbcode/`** - BBCode formatted files for forum posting

### Distribution Files / 分发文件

Zip files for distribution (should be in root or separate distribution folder):
- `report.zip` - Contains `report_en.md` and `report_zh.md`
- `suggestion.zip` - Contains `suggestion_en.md` and `suggestion_zh.md`
- `technical implementation.zip` - Contains `technical implementation_en.md` and `technical implementation_zh.md`

分发用压缩包（应在根目录或单独的分发文件夹中）：
- `report.zip` - 包含 `report_en.md` 和 `report_zh.md`
- `suggestion.zip` - 包含 `suggestion_en.md` 和 `suggestion_zh.md`
- `technical implementation.zip` - 包含 `technical implementation_en.md` 和 `technical implementation_zh.md`
