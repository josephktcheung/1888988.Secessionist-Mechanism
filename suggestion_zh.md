# 分离主义机制改进建议

**核心假设：** 分离主义叛军是希望脱离控制者的民族主义叛军。**如果存在主要文化国家并获得干预权：** 叛军在胜利后加入该国成为分离主义属国，该国成为战争领袖。**如果不存在主要文化国家或无人获得干预权：** 叛军成为独立国家并自行领导战争，在宿敌国家的支持下赢得独立。**历史案例：** 希腊独立战争（1821年）——拜占庭于1453年被摧毁，因此不存在希腊主要文化国家。希腊叛军独立并在俄罗斯、不列颠和法兰西（奥斯曼的宿敌）的支持下赢得战争。详见案例研究 1D。

## 现有先例：EU5 中已验证的机制

EU5 已有多个可适用于分离主义叛乱的已验证机制：

1. **奥斯曼崛起 Press_Claims 机制**：允许国家在局势框架内使用杜卡特获取宣战理由。`press_claims` 行动（type = situation）表明局势框架可以处理基于杜卡特的获取，国家可以评估并选择是否获取（无强制参与），成本可以根据经济因素（scaled_gold）进行缩放。

2. **德里陷落局势**：展示了基于优先级的可见性（标签、邻国、属国、敌人/宿敌、地区），使用 `every_country` 限制的严格过滤，用于跟踪参与者的全局变量列表，以及交错的事件通知。

3. **意大利战争局势**：展示了使用 `ordered_country` 进行复杂优先级计算（军事力量、等级、收入），多层级通知，以及战略利益评估。**关键**：在限制子句中使用 `is_subject = no` 排除属国排名（第 139、185、254、299、352、404 行），确立了在排名系统中取消属国资格的先例。

4. **干预属国内战**：展示了国家如何使用 `join_war_as_attacker` 或 `join_war_as_defender`（`reason = intervene`）加入现有内战。这正是加入分离主义叛乱所需的机制。

5. **红巾军叛乱局势**：展示了基于距离/关系的过滤，使用严格限制的 `every_country`，以及授予加入战争能力的局势行动（`rtr_join_chinese_defense` 使用 `join_war_as_attacker`/`join_war_as_defender`）。

6. **包围网机制**：包围网（`game/in_game/common/international_organizations/coalition.txt`）具有 `has_leader_country = yes` 和 `leader_type = country`。包围网领袖通过成员中最高 `great_power_score` 自动选择（未定义明确的 `leader_change_method`，因此游戏默认选择最强成员）。当包围网目标参与时，包围网成员可以通过 `join_defensive_wars_always` 和 `join_offensive_wars_always` 加入战争。这允许包围网领袖协调包围网干预，同时防止级联升级（只有包围网领袖可以触发包围网参与）。干预权系统与这些现有包围网机制集成，允许包围网领袖获取干预权并协调包围网成员对抗侵略性大国。

干预权系统提案直接建立在这些已验证的基础上，将其适用于分离主义叛乱，并增加了雇佣兵支持、基于优先级的通知系统，以及与包围网机制的集成等额外功能。

## 问题总结

当前分离主义机制存在多个关键问题：

1. 强制参与战争，无拒绝选项
2. **原所有者被完全忽视**——系统仅关注文化主导地位，忽略历史所有权。例如：在拜占庭案例中，奥斯曼（叛乱领土的原所有者）被完全忽视，而优先选择埃雷特纳（文化主导但非原所有者）。原所有者没有机会收回其原有领土。
3. 强制参与战争时触发破坏停战协议的惩罚
4. 强制参与战争时联合统治破裂（已验证：当西西里叛军出现时，法兰西-西西里联合统治破裂）
5. 荒谬的距离支持请求（例如，元 支持 6000+ 公里外的东欧叛军）
6. 宗主被迫参与且无法控制
7. 缓慢的干预授权创建使及时干预变得不可能（拜占庭需要 65 个月，每月 0.31，才能建立 20 间谍网强度以对抗奥斯曼支持叛军，但到那时叛乱可能已经被镇压）
8. 支持叛军行动很少使用（拜占庭需要 97 个月，每月 0.31）
9. 叛军军事薄弱（只有征召兵，没有正规军），无法在参战国派遣军队之前存活足够长的时间
10. 世界大战风险 - 多个国家可以加入，不必要地升级冲突
11. **通过不对称同盟调用导致的世界大战升级**——单个局部叛乱可以升级为世界大战。防御方可以调用所有盟友，而被迫调用的进攻方无法调用自己的盟友，创建危险的升级机制，防御方可以系统地利用分离主义叛乱不对称地攻击敌人的同盟集团（见案例研究 1C：波兰叛乱升级）

## 案例研究 1：波萨德叛乱 - 强制参与问题

**场景：** 1337年的元（中华帝国）。波萨德（黑海附近）原由金帐汗国（与莫斯科接壤的蒙古国家）拥有。莫斯科征服了波萨德。1,669 名蒙古人开始分离主义叛乱。叛乱寻求元（6000+ 公里外）的支持，忽略了金帐汗国（原所有者、邻国）这一合理选择。

**当前游戏行为：** 游戏将元识别为占主导地位的蒙古文化国家。元在第 1 天被**强制**加入，在 6000+ 公里外作战，**无选择权**。从波萨德到大都（北京）的信使单程需要 9-18 个月（往返 3-4 年），使得第 1 天参与在物理上不可能。

**逻辑问题：** 蒙古叛军应该**首先**考虑原所有者（金帐汗国）。当前系统**完全忽略原所有权**——仅关注文化主导地位，忽略历史主张和逻辑优先级。

**新系统如何修复：** 排名公式给予**原所有者奖励：100 分**（最高优先级）。金帐汗国（原所有者 + 邻国）排名最高；元（6000+ 公里外）排名较低。**这恢复了逻辑优先级：原所有者 > 邻国 > 文化主导**。同时修复了缓慢的宣战理由创建（65 个月 → 在 80-90% 时立即获取）和叛军军事弱点（杜卡特在革命前资助雇佣兵）。

## 解决方案 1：简化拒绝和预评估

1. 简单拒绝机制：被迫进入内战时弹出窗口，选项支持/拒绝，如果拒绝则无惩罚
2. 豁免强制参与的惩罚：添加 `forced_civil_war_participant` 标志，在应用破坏停战协议的惩罚和联合统治破裂之前检查
3. 预评估系统：距离检查（>2000 = 不调用）、土地欲望检查、军事通行权 可行性，优先级：原所有者 > 邻国 > 文化主导
4. 宗主 战争领袖 控制：当被拖入属国内战时，使宗主成为 战争领袖
5. 军事通行权 例外：允许 战争领袖 在防御战争期间向盟友请求 军事通行权

## 解决方案 1B：使用现有加入战争机制的临时解决方案

**临时方法：** 此解决方案可以在完整干预权系统之前快速实施，使用现有的 `can_join_offensive_war_with` 和 `can_join_defensive_war_with` 触发器（类似于红巾军叛乱的 `rtr_join_chinese_defense` 行动）。这提供了即时修复，同时开发综合系统。

**核心概念：**

不是强制参与，系统使用现有触发器检查国家是否可以加入内战，然后给他们一个可选事件加入（类似于红巾军叛乱的工作方式 - 无强制参与，国家可以选择加入或拒绝）。

**关键特性：**

1. **原所有者优先级：** 当分离主义叛乱爆发时，系统在检查文化主导国家之前首先检查原所有者（使用 `previous_owner`）。这修复了拜占庭案例，其中奥斯曼（原所有者）应该优先于埃雷特纳（文化主导但非原所有者）。

2. **可选参与：** 国家在内战开始时收到事件通知，选项为"加入战争"或"拒绝"。无强制参与 - 国家可以拒绝而不受惩罚（如红巾军叛乱）。

3. **资格检查：** 使用现有的 `can_join_offensive_war_with` 和 `can_join_defensive_war_with` 触发器检查国家是否实际可以加入战争（距离、外交关系、停战协议等）。只有符合条件的国家收到通知。

4. **优先级系统：** 通知优先级：原所有者（相同文化）> 邻国（相同文化）> 文化主导（相同文化）> 宿敌。这确保了逻辑优先级，同时使用现有游戏机制。

5. **无资源成本：** 如红巾军叛乱，加入无 杜卡特 成本。这比完整的干预权系统更简单，但仍修复了核心问题。

6. **宗主选项：** 宗主收到特殊通知，可以选择加入或拒绝（解决漏洞 1）。如果怀疑分数高，宗主可以使用现有的 `cancel_subject` 机制立即取消附庸地位。

**工作原理：**

1. 当分离主义叛军达到 100% 且内战开始时（现有 EU5 行为）

2. 系统按优先级顺序检查符合条件的国家：
- 层级 1：具有相同文化的原所有者（使用 `previous_owner` 检查）
- 层级 2：具有相同文化的邻国
- 层级 3：文化主导国家（如果尚未选择）
- 层级 4：防御方的 宿敌

- 对于每个符合条件的国家，系统检查：
- `can_join_offensive_war_with = rebel_country`（他们可以加入 进攻方 一方吗？）
- `NOT = { is_at_war_with = defender }`（未处于战争中）
- `NOT = { has_truce_with = defender }`（无停战协议）
- `distance_to_capital <= 5000`（合理距离）
- 其他标准检查（未破产、未处于内战中等）

- 符合条件的国家收到事件通知："分离主义叛乱已在 [DEFENDER] 爆发。您想加入叛军一方的战争吗？"

- 选项：
- "加入战争" - 国家使用 `join_war_as_attacker`（`reason = intervene`）加入内战的 进攻方 一方（如 `intervene_in_subject_civil_war`）
- "拒绝" - 国家拒绝，无惩罚，不参与

- 宗主收到特殊通知，包含额外选项：
- "加入战争" - 加入防御战争
- "拒绝 - 让附庸独自战斗" - 拒绝加入，附庸独自战斗
- "取消 附庸 地位" - 如果怀疑分数高，使用 `cancel_subject` 立即取消 附庸 地位

**优势：**

- 快速实施 - 使用现有触发器和机制（无需新的 局势 框架）
- 修复强制参与问题 - 国家可以拒绝
- 修复原所有者优先级 - 在 拜占庭 案例中，奥斯曼 优先于 埃雷特纳
- 使用已验证的机制 - `can_join_offensive_war_with` 已经平衡和测试
- 无资源成本 - 比完整系统更简单，但仍然有效
- 解决漏洞 1 - 宗主可以拒绝或取消 附庸 地位
- 防止世界大战 - 资格检查自然限制参与

**限制（与完整干预权系统相比）：**

- 无早期警告（80-90%）- 国家仅在叛乱爆发时收到通知
- 无 雇佣兵 资助 - 叛军不接收用于 雇佣兵 的 杜卡特
- 无预获取系统 - 国家无法提前准备干预权
- 战略深度较低 - 无 杜卡特 投资，无支持叛军协同

**推荐使用：**

此临时解决方案可以作为阶段 0.5 或阶段 1.5 实施 - 在即时热修复之后，但在完整干预权系统之前。它在开发综合系统的同时提供即时缓解强制参与。

## 解决方案 2：限制资格的干预权系统

**核心概念：**

**重要术语澄清**：此系统**不**授予传统的宣战理由（CB）来宣战。相反，它授予"干预权"，允许国家在分离主义叛军爆发时加入现有内战。当分离主义叛军达到 100% 进度时，它们自动对防御方发动内战。已获得干预权的国家可以加入此现有内战的叛军一方，类似于国家在 EU5 中可以加入正在进行的战争（使用 `join_war_as_attacker`，如 `intervene_in_subject_civil_war`）。

当分离主义叛军每月进展时，它模拟建立间谍网并与叛军建立联系所需的时间投资、外交官资源和每月支出。当叛军达到 80-90% 时，符合条件的国家会收到干预权可用的通知。国家可以通过花费杜卡特（发送给叛军的货币支持）来获得此权利。这些杜卡特允许叛军在革命开始前雇佣雇佣兵，给予他们军事力量以在最初的政府镇压中生存，并坚持到参战国可以派遣军队。

当叛乱爆发时（叛军达到 100%），它创建内战。已获得干预权的国家可以通过派遣 1 名外交官，使用 `join_war_as_attacker`（`reason = intervene`）加入此现有内战（如 `intervene_in_subject_civil_war`）。这与宣战不同——这是加入叛军一方的现有冲突。

此系统直接建立在已验证的先例之上：
- **奥斯曼崛起 press_claims**：在 局势 框架中基于 杜卡特 的获取
- **德里陷落**：基于优先级的可见性和严格过滤
- **意大利战争**：使用 `ordered_country` 进行复杂优先级计算，自定义评分
- **干预属国内战**：使用 `join_war_as_attacker` 加入现有内战
- **红巾军叛乱**：授予加入战争能力的 局势 行动

## 资格限制（防止世界大战）

**关键限制**：为防止干预权获取升级为世界大战，系统将资格限制为两个逻辑类别：

**类别 1：排名最高的相同主要文化国家（优先级提供系统）**
- 与叛乱人口具有相同主要文化的国家
- **属国取消资格**：属国（附庸、分离主义属国、采邑、联合统治 初级伙伴等）**被取消**类别 1 排名资格。这防止了遥远地区的小型分离主义属国（例如，蒙古地区的蒙古文化分离主义属国）被错误地优先考虑用于远距离的叛乱（例如，在格鲁吉亚）。这也防止属国获取会导致它们加入针对其宗主的战争的干预权，这会破坏它们的属国关系（例如，当西西里叛军出现在 法兰西 控制的领土时，法兰西-西西里 联合统治 破裂）。只有具有相同主要文化的独立国家才有资格进行类别 1 排名。
- **关键限制 - 优先级系统**：只有排名最高的相同文化国家最初获得机会（类似于"优先拒绝权"）。如果他们拒绝或在时间限制内（例如，30 天）未获取，下一个排名的相同文化国家变得符合条件。这创建了一个基于优先级的顺序系统，防止多个相同文化国家同时获取，同时确保最佳候选者获得首次机会。
- 排名公式：`(original_owner_bonus * 100) + (neighbor_bonus * 50) + (cultural_influence * 10) + (country_rank_level * 5) - (distance_to_capital / 10)`
- 示例 1：对于波萨德的蒙古叛军，金帐汗国（原所有者 + 邻国，分数：`100+50+... = 高`）排名高于 元（距离远，文化影响力高但距离惩罚，分数：`0+0+563*10-6000/10 = 较低`）。金帐汗国 获得首次机会。如果 金帐汗国 拒绝，元 变得符合条件。
- 示例 2：**关键**：对于前 奥斯曼 领土中的土耳其叛军（拜占庭 案例），奥斯曼（原所有者，分数：`100+... = 高`）排名高于 埃雷特纳（非原所有者，分数：`0+cultural_influence*10+... = 较低`）。奥斯曼 获得首次机会。如果 奥斯曼 拒绝，埃雷特纳 变得符合条件。这修复了当前系统的缺陷，其中 奥斯曼 被完全忽视，而优先选择文化主导的 埃雷特纳。
- 示例 3：**属国取消资格**：对于 法兰西 控制领土中的西西里叛军，西西里（文化主导但是 法兰西 的 联合统治 初级伙伴）**被取消**类别 1 排名资格。只有独立的西西里文化国家（如果存在）才有资格。这防止了当 西西里 本来会被迫加入针对其高级伙伴 法兰西 的战争时，联合统治 破裂。类似地，小型分离主义属国（例如，蒙古地区的蒙古文化分离主义属国）被取消优先考虑远距离叛乱的资格（例如，在格鲁吉亚），防止错误的远距离干预。
- **先例**：这遵循了 EU5 的既定模式。意大利战争 局势（`italian_wars.txt`）在限制子句中使用 `ordered_country` 和 `is_subject = no` 来排除属国排名（第 139、185、254、299、352、404 行）。类似地，奥斯曼崛起（`rise_of_the_ottomans.txt`）使用 `limit = { is_subject = no }` 排除属国排名（第 48 行）。海盗黄金时代（`golden_age_of_piracy.txt`）也从海盗国家排名中排除属国（第 82 行）。宗教战争（`war_of_religions.txt`）从神圣罗马帝国成员检查中排除属国（第 20、24 行）。这种模式在多个 局势 中是一致的。
- 实施：使用 `ordered_country` 和 `order_by` 评分对所有相同文化国家进行排名，**排除属国**（在限制子句中使用 `is_subject = no`，遵循意大利战争先例）。只有排名最高的独立国家最初收到通知。如果他们拒绝或时间到期，下一个排名的独立国家收到通知。
- 优先级机制：系统跟踪哪些相同文化国家已获得机会。如果排名最高的拒绝或在时间限制内未响应，系统将机会扩展到下一个排名的国家。这按顺序继续，直到有人获取或所有符合条件的国家拒绝。
- 这确保只有**一个**相同文化国家可以同时获取，防止文化集团全部加入，同时优先考虑最合理的候选者（原所有者，然后是邻国，然后是文化主导）
- **主要文化国家的停战协议例外**：主要文化国家即使与防御方有停战协议也可以获取干预权。但是，为防止利用系统，**应实施以下成本/惩罚机制之一：**
- **选项 A - 更高成本（推荐）**：有停战协议的主要文化国家支付更高成本（例如，2.0x 乘数而不是基础成本）。这代表打破停战协议以支持叛军的额外外交和经济成本。更高的成本防止玩家通过反复打破停战协议以利用系统，同时在价值证明成本合理时仍允许战略干预。
- **选项 B - 减少破坏停战协议的惩罚**：有停战协议的主要文化国家在加入战争时仍会收到破坏停战协议的惩罚，但严重程度降低（例如，-25 稳定度而不是 -50，+0.5 厌战度而不是 +1，+12 敌意而不是 +25）。这承认支持自己文化的独立斗争是一种不对称战争形式，不会完全违反停战协议的精神，同时仍对打破它施加后果。
- **选项 C - 组合方法**：更高成本（1.5x 乘数）和减少的破坏停战协议的惩罚（50% 严重程度）。这提供了最强的防利用保护，同时仍允许战略干预。
- 允许停战协议购买的理由：
- 支持自己文化的独立斗争是一种不对称战争形式，不会完全违反停战协议的精神
- 停战协议可能是被迫的（他们输掉了战争），支持叛军是抵抗的合法延续
- 如果国家在停战协议期间之前/期间使用了"支持叛军"行动，购买干预权是该支持的延续
- 这是一个有意识的战略决策（与强制参与不同），允许国家选择是否打破停战协议以支持其文化的独立，但需要适当的成本/惩罚
- 防止防御方利用分离主义叛乱迫使没有选择权的战败方打破停战协议
- 示例：波兰与俄罗斯有停战协议（来自俄罗斯征服波兰领土的先前战争）。当波兰分离主义叛乱在俄罗斯控制的领土上爆发时，波兰可以获取干预权以支持叛乱，但必须支付更高成本（选项 A）或接受减少的破坏停战协议的惩罚（选项 B）。这代表波兰做出战略选择以打破停战协议来收回波兰领土，接受经济或外交成本，而不是被迫这样做而没有后果。

**类别 2：防御方的 宿敌（全部符合条件）**
- 防御方的 宿敌 国家
- 这些国家有削弱防御方的战略利益
- 示例：如果 奥斯曼 正在防御希腊叛军，奥斯曼 的 宿敌（例如，奥地利、俄罗斯、威尼斯）可以获取干预权
- 严格检查：`is_rival_of = defender`
- 理由：宿敌 在支持针对其敌人的叛乱方面有真正的战略利益
- 更高成本：1.5x 乘数（因为他们缺乏文化联系）

**类别 3：针对防御方的 包围网 领袖**
- 针对防御方的 包围网 领袖 国家
- **关键限制 - 仅 包围网 领袖**：只有 包围网 领袖可以代表 包围网 购买干预权。这防止每个 包围网 成员单独购买，从而造成不受控制的升级。包围网 领袖代表 包围网 的集体战略利益。
- 严格检查：`is_leader_of_international_organization`，其中组织是 包围网，且 `is_target_of_international_organization_of_type = { type = coalition target = defender }`（防御方是 包围网 目标）。注意：包围网 领袖通过成员中最高 `great_power_score` 自动选择，因此通常最强的 包围网 成员成为领袖。
- 理由：包围网 是针对侵略扩张形成的（敌意 > 50）。当分离主义叛乱在 包围网 目标的领土上爆发时，包围网 成员有支持叛乱以削弱侵略性大国的战略利益。但是，只有 包围网 领袖可以协调此支持以防止级联升级。
- 更高购买成本：1.5x 乘数（与 宿敌 相同，因为他们缺乏文化联系但有战略利益）
- 包围网 成员可以通过 包围网 机制加入：当 包围网 领袖购买干预权并加入战争时，包围网 成员可以通过现有 包围网 机制被调用（`coalition.txt` 中的 `join_defensive_wars_always` 和 `join_offensive_wars_always`）。但是，只有 包围网 领袖（购买干预权的）可以调用 包围网 成员 - 个别 包围网 成员无法调用自己的盟友，防止级联升级。
- 示例：如果 俄罗斯 有一个针对它的 包围网（由于侵略扩张而形成），波兰分离主义叛乱在 俄罗斯 领土上爆发，包围网 领袖（例如，奥地利 - 通过最高 `great_power_score` 自动选择为最强 包围网 成员）可以购买干预权。当 奥地利 加入战争时，包围网 成员（例如，瑞典）可以通过 包围网 机制被调用，但只有 奥地利（包围网 领袖）可以调用它们 - 瑞典 无法调用自己的盟友。注意：普鲁士 是 俄罗斯 的盟友，不在 包围网 中。

**为什么这些限制防止世界大战：**
- 将参与限制为有真正利益的国家（文化联系或战略竞争或包围网成员身份）
- 只有**一个**相同文化国家可以加入（通过公式排名最高），防止文化集团
- 宿敌 自然受限（国家通常有 2-3 个 宿敌）
- 包围网 参与受控：只有 包围网 领袖（通过成员中最高 `great_power_score` 自动选择）可以获取干预权，只有 包围网 领袖可以调用 包围网 成员，防止每个 包围网 成员调用所有盟友
- 防止随机国家仅因为可以而加入
- 保持冲突本地化和历史合理
- 仍允许历史场景：排名最高的相同文化国家支持其亲属，宿敌 利用机会削弱敌人，包围网 协调对抗侵略性大国
- 包围网 集成：使用现有 包围网 机制（`join_defensive_wars_always` 和 `join_offensive_wars_always`），但限制谁可以触发 包围网 参与（只有获取干预权的 包围网 领袖）。包围网 领袖通过成员中最高 `great_power_score` 自动选择（如 `coalition.txt` 中定义的 `has_leader_country = yes` 和 `leader_type = country`），确保最强的 包围网 成员协调干预，防止级联升级

## 它如何解决问题

**蒙古案例研究：**
- 金帐汗国（原所有者，邻国）通过公式排名最高：`(100 原所有者 + 50 邻国 + ...) = 最高分`
- 元（6000+ 公里外）排名较低：`(0 原所有者 + 0 邻国 + 563*10 文化影响力 - 6000/10 距离 = 负数)`
- 金帐汗国 是排名最高的相同文化国家，当叛军达到 80% 时立即收到通知
- 金帐汗国 可以评估并获取干预权（或拒绝，无惩罚）
- 如果 金帐汗国 获取：玩家支付 杜卡特，叛军雇佣 雇佣兵（14+ 天集结），玩家获得干预权。元 永远不会收到通知（优先级系统在 金帐汗国 处停止）。
- 如果 金帐汗国 拒绝或在 30 天内未响应：元（下一个排名）变得符合条件并收到机会。元 然后可以获取或拒绝。
- 当叛乱爆发时（叛军达到 100%），内战自动开始。**如果 金帐汗国 获取了干预权：** 叛军国家自动加入 金帐汗国 作为分离主义属国，金帐汗国 成为 进攻方 战争领袖（不是叛军国家/附庸）。这防止了 附庸 利用，其中 附庸 在宗主之前获得所有利益。金帐汗国 然后可以调用其盟友并控制和平谈判。
- **如果没有主要文化国家获取干预权：** 叛军国家独立并领导战争。获取干预权的 宿敌 仍可以加入，但叛军国家保持独立。**历史案例：** 希腊独立战争（1821年）- 拜占庭 于 1453 年被摧毁，因此不存在希腊主要文化国家。希腊叛军独立并在 俄罗斯、不列颠和 法兰西（奥斯曼 的 宿敌）的支持下赢得战争。详见案例研究 1D。
- 获取干预权的国家然后可以使用 `join_war_as_attacker`（如 `intervene_in_subject_civil_war`）加入此现有内战的叛军一方
- 此优先级系统确保只有一个相同文化国家可以同时获取，防止多个国家同时全部加入，同时优先考虑最合理的候选者

**拜占庭-奥斯曼 案例研究（原所有者优先级恢复）：**

**当前系统缺陷：** 奥斯曼（叛乱领土的原所有者）被**完全忽视**，因为他们不是文化主导的土耳其国家。埃雷特纳（文化主导）被迫加入，即使埃雷特纳可能与这些特定领土没有历史联系。

**新系统修复：**
- 奥斯曼（原所有者）通过公式排名最高：`(100 原所有者 + ...) = 最高分`
- 埃雷特纳（非原所有者）排名较低：`(0 原所有者 + 文化影响力 * 10 + ...) = 较低分`
- 奥斯曼是排名最高的相同文化国家，当叛军达到 80% 时立即收到通知
- 奥斯曼可以评估并获取干预权以收回其原有领土（或拒绝，无惩罚）
- 如果奥斯曼购买：玩家支付杜卡特，叛军雇佣雇佣兵，玩家获得干预权以收回前奥斯曼领土。埃雷特纳永远不会收到通知（优先级提供系统在奥斯曼处停止）。
- 如果奥斯曼拒绝或在 30 天内未响应：埃雷特纳（下一个排名）变得符合条件并收到购买报价。埃雷特纳然后可以购买或拒绝。
- 当叛乱爆发时（叛军达到 100%），内战自动开始。**如果奥斯曼购买了干预权：** 叛军国家自动加入奥斯曼作为分离主义属国，奥斯曼成为进攻方战争领袖（不是叛军国家/附庸）。这防止了附庸利用，并确保奥斯曼控制和平谈判以收回前奥斯曼领土。奥斯曼然后可以调用其盟友并控制战争。
- **如果没有主要文化国家购买干预权：** 叛军国家独立并领导战争。购买干预权的宿敌仍可以加入，但叛军国家保持独立。**历史案例：** 希腊独立战争（1821年）——拜占庭于 1453 年被摧毁，因此不存在希腊主要文化国家。希腊叛军独立并在俄罗斯、不列颠和法兰西（奥斯曼的宿敌）的支持下赢得战争。详见案例研究 1D：希腊独立战争的详细分析。
- 购买干预权的国家然后可以使用 `join_war_as_attacker` 加入此现有内战的叛军一方以收回领土
- 此优先级提供系统确保原所有者获得首次机会，解决原所有者被忽视的问题
- **这恢复了逻辑优先级：原所有者（奥斯曼）> 文化主导（埃雷特纳），顺序提供确保一次只有一个可以购买**

**世界大战升级问题 - 新系统如何修复：**
- **当前系统缺陷：** 创建比历史第一次世界大战/第二次世界大战更糟糕的不对称同盟调用。防御方可以调用所有盟友，但被迫调用的进攻方无法调用自己的盟友，创建单边升级。
- **历史先例：** 第一次世界大战（萨拉热窝刺杀）和第二次世界大战（但泽危机）展示了区域冲突如何通过同盟系统升级。但是，在这两种情况下，双方都可以在承诺之前与盟友协调——国家在其决策中有代理权。
- **新系统修复：** 优先级系统使参与可选。国家可以在获取干预权之前评估并与盟友协调。如果排名最高的国家拒绝，不会发生升级。如果他们获取，双方都可以与盟友协调（如历史世界大战），创建平衡升级而不是不对称单边战争。
- **资格限制：** 只有排名最高的相同文化国家（通过优先级系统）或防御方的宿敌可以获取，防止随机国家加入并将升级限制为有真正利益的国家。
- **防止强制升级：** 国家可以拒绝购买，防止自动世界大战升级。这与历史现实匹配，其中并非所有区域事件都会升级（例如，1912-1913 年巴尔干战争直到萨拉热窝才升级为第一次世界大战）。
- **平衡升级：** 如果发生升级，双方都可以在承诺之前与盟友协调，匹配历史世界大战，其中国家在其决策中有代理权。

## 案例研究 1C：波兰叛乱升级 - 世界大战场景

**历史先例：区域冲突如何升级为世界大战**

历史表明，区域冲突可以通过 同盟 系统升级为世界大战。两个主要例子：

1. **第一次世界大战（1914年）：** 萨拉热窝刺杀弗朗茨·斐迪南大公（1914年6月28日）是巴尔干地区的区域事件。奥匈帝国对塞尔维亚的最后通牒通过 同盟 系统升级：俄罗斯 动员支持塞尔维亚，德意志 支持奥匈帝国，法兰西 支持俄罗斯，不列颠 进入支持法兰西。单个区域事件升级为涉及所有主要大国的全球战争。

2. **第二次世界大战（1939年）：** 德意志 入侵波兰（1939年9月1日）由但泽危机引发 - 关于单个城市的区域争端。但是，不列颠和法兰西已保障波兰独立。当德意志入侵时，不列颠和法兰西宣战，拉入其盟友。一个城市的区域冲突升级为世界大战。

**EU5 场景（18世纪后期 - 瓜分时代）：**

在 EU5 的时间范围内，可能发生类似的升级场景：

- 形成了两个主要同盟集团（类似于第一次世界大战/第二次世界大战风格）：
- **同盟集团 A（防御方）：** 俄罗斯（防御方）、普鲁士
- **同盟集团 B（潜在进攻方）：** 法兰西、奥斯曼、瑞典
- 俄罗斯在之前的战争中从波兰-立陶宛联邦征服了波兰领土
- 波兰与俄罗斯有停战协议（来自俄罗斯征服波兰领土的先前战争）
- 波兰仍然存在（文化主导的波兰国家），但**未**与法兰西、奥斯曼或瑞典结盟，且**未**在针对俄罗斯的包围网中
- 俄罗斯增长过于激进，形成了针对它的包围网（包围网成员：奥地利（领袖）、瑞典等）。波兰**不在**此包围网中——它与俄罗斯有停战协议，不是包围网的一部分。
- 小型波兰分离主义叛乱在单个地区爆发（例如，波多利亚或沃里尼亚）——类似于但泽或萨拉热窝是区域热点
- 叛乱仅涉及一个省份的数千名波兰叛军
- 法兰西**保障**了波兰-立陶宛的独立（现有游戏机制：`guarantee` 关系，`called_in_defensively = giving`）
- 法兰西也是俄罗斯的宿敌（削弱俄罗斯的战略利益）
- 奥地利是针对俄罗斯的包围网领袖（由于俄罗斯的侵略扩张而形成的包围网）——奥地利被自动选为包围网领袖，因为它在包围网成员中拥有最高的 `great_power_score`

**当前游戏行为 - 世界大战升级（如第一次世界大战/第二次世界大战）：**
1. 波兰分离主义叛乱在俄罗斯控制的领土爆发（区域事件，如萨拉热窝或但泽）
2. 波兰（文化主导的波兰国家）在第 1 天被**强制**加入叛乱一方（进攻方）——**无选择权**，就像国家通过同盟义务被拉入第一次世界大战/第二次世界大战一样。**关键：** 波兰与俄罗斯有停战协议，但仍被迫加入，触发破坏停战协议的惩罚（-50 稳定度，+1 厌战度，+25 敌意），即使它没有选择。
3. **关键不对称（与第一次世界大战/第二次世界大战同盟系统不同）：**
- **俄罗斯（防御方）可以调用所有盟友：** 俄罗斯调用普鲁士加入防御战争（如第一次世界大战中德意志调用奥匈帝国，或第二次世界大战中不列颠调用法兰西）。俄罗斯和普鲁士现在对抗波兰。
- **波兰（被迫调用的进攻方）无法调用其盟友：** 波兰是进攻方，因此即使它有盟友，也无法调用它们加入（与第一次世界大战/第二次世界大战不同，双方都可以与盟友协调）。波兰独自对抗三个主要大国。
- 结果：涉及数千名叛军的局部叛乱升级为重大战争：
- 俄罗斯 + 普鲁士（2 个主要大国）vs 波兰（1 个国家，独自）
- 波兰 没有选择 - 它被迫进入战争，无法调用自己的盟友
- 战争变得严重一边倒，波兰-立陶宛面临压倒性劣势
- 这**比**历史第一次世界大战/第二次世界大战**更糟糕**，在历史世界大战中双方都可以与盟友协调 - 当前系统创造了更加不对称的升级
- 历史比较：在第一次世界大战/第二次世界大战中，双方都可以在承诺之前与盟友协调。当前 EU5 系统阻止被迫调用的 进攻方 协调，创造了比历史世界大战更糟糕的不对称升级。

**新系统行为 - 结合所有机制的综合场景：**
1. 波兰分离主义叛乱达到 80-90% 进度（早期警报系统）
2. 优先级系统：波兰（排名最高的相同文化国家）首先获得机会。波兰 与 俄罗斯 有 停战协议，但仍可以获取（必须支付更高成本 2.0x 或接受减少的惩罚 50% 严重程度）。
3. **场景 A - 波兰获取：** 获取干预权。当叛乱爆发时，叛军国家加入波兰作为分离主义属国，波兰成为进攻方战争领袖。**注意：** 法兰西保障了波兰，但保障机制（`called_in_defensively = giving`）只在被保障的国家被攻击时（防御方）触发，而不是在被保障的国家攻击时（进攻方）触发。由于波兰是进攻方，法兰西的保障不会触发。如果法兰西想加入战争，它必须购买干预权（类别 2 - 宿敌）以作为进攻方战争参与者加入，成本是杜卡特（1.5x 倍数）而不是仅 1 个外交官。然而，如果法兰西购买干预权，它作为进攻方战争参与者加入（不是战争领袖），因此不能调用盟友。俄罗斯（防御方战争领袖）调用普鲁士。波兰（进攻方战争领袖）可以调用其自己的盟友（如果有）。结果：如果法兰西购买干预权，平衡升级（俄罗斯+普鲁士 vs 波兰+法兰西+叛军+波兰的盟友）。只有波兰和俄罗斯可以调用盟友（防止级联）。如果法兰西不购买，更简单的升级（俄罗斯+普鲁士 vs 波兰+叛军+波兰的盟友）。**关键洞察：** 保障机制更便宜（1 个外交官）但仅在防御时有效。干预权更昂贵（杜卡特）但允许作为进攻方加入，尽管除非成为战争领袖否则无法调用盟友。
4. **场景 B - 波兰拒绝，法兰西获取：** 法兰西（宿敌）获取。当叛乱爆发时，叛军国家独立（无主要文化国家获取）。法兰西加入但叛军国家领导战争。俄罗斯调用普鲁士。结果：受控升级（俄罗斯+普鲁士 vs 法兰西+独立叛军）。展示了独立国家形成（如 1821 年希腊独立战争——不存在希腊主要文化国家，叛军独立）。
5. **场景 C - 两者都拒绝，包围网领袖获取：** 奥地利（针对俄罗斯的包围网领袖）获取。包围网成员（瑞典）通过包围网机制加入。叛军国家独立。俄罗斯调用普鲁士。结果：包围网协调（俄罗斯+普鲁士 vs 奥地利+瑞典+独立叛军）。只有包围网领袖可以调用包围网成员（防止级联）。
6. **场景 D - 波兰和法兰西都获取：** 波兰（类别 1 - 相同文化）通过优先级系统首先获取。法兰西（类别 2 - 宿敌）也获取（不同类别，无限制）。当叛乱爆发时，叛军国家加入波兰作为分离主义属国，波兰成为进攻方战争领袖。法兰西使用其购买的干预权作为进攻方战争参与者加入。俄罗斯（防御方战争领袖）调用普鲁士。波兰（进攻方战争领袖）可以调用其自己的盟友（如果有）。结果：多个干预权（俄罗斯+普鲁士 vs 波兰+法兰西+叛军+波兰的盟友）。只有波兰和俄罗斯可以调用盟友（防止级联）。展示了不同类别（类别 1 相同文化和类别 2 宿敌）可以同时购买，但只有类别 1 购买者在叛军加入作为属国时成为战争领袖。
7. **场景 E - 波兰、法兰西和奥地利都获取：** 波兰（类别 1 - 相同文化）通过优先级系统首先获取。法兰西（类别 2 - 宿敌）也获取。奥地利（类别 3 - 针对俄罗斯的包围网领袖）也获取（所有三个类别可以同时购买）。当叛乱爆发时，叛军国家加入波兰作为分离主义属国，波兰成为进攻方战争领袖。法兰西使用其购买的干预权作为进攻方战争参与者加入。奥地利使用其购买的干预权作为进攻方战争参与者加入，并可以通过包围网机制调用包围网成员（瑞典）。俄罗斯（防御方战争领袖）调用普鲁士。波兰（进攻方战争领袖）可以调用其自己的盟友（如果有）。结果：所有三个类别的最大升级（俄罗斯+普鲁士 vs 波兰+法兰西+奥地利+瑞典+叛军+波兰的盟友）。只有波兰、奥地利（用于包围网成员）和俄罗斯可以调用盟友（防止级联）。展示了所有三个类别可以同时购买，但只有类别 1 购买者成为战争领袖，只有包围网领袖可以调用包围网成员。
8. **场景 F - 全部拒绝：** 不发生升级。局部叛乱保持局部（历史准确——并非所有区域事件都会升级）。
9. 展示的关键机制：优先级系统、停战协议例外、保障机制、包围网集成、独立国家形成、受控升级、附庸利用预防（购买者成为战争领袖）、不同类别的多个干预权、所有三个类别的最大升级场景。

**新系统如何解决这些问题：** 缓慢的宣战理由创建（65 个月 → 在 80-90% 时立即获取）、叛军军事弱点（杜卡特 在革命前资助 雇佣兵）、支持叛军行动变得有价值（加快进度 + 实现及时获取）。当叛乱爆发时，国家可以立即使用已获取的干预权加入。

## 通知系统

**层级 1：排名最高的相同文化国家** - 只有排名最高的国家最初获得机会（优先拒绝权）。如果拒绝，下一个排名的国家变得符合条件。顺序系统确保一次只有一个相同文化国家可以获取。使用带排名公式的 `ordered_country`。

**层级 2：防御方的 宿敌** - 所有 宿敌 在 85-90% 时收到通知（交错）。更高成本（1.5x 乘数）。严格检查：`is_rival_of = defender AND NOT = { culture = rebel_culture }`。

## 通知控制机制

**游戏设置：** 五个选项："所有通知"（默认）、"仅相同文化"（约 70-80% 减少）、"仅 宿敌"（约 60-70% 减少）、"仅原所有者"（约 85-95% 减少）、"禁用"。

**冷却系统：** 每个国家冷却（6 个月），每个叛乱冷却（5% 进度）。检查：`NOT = { has_variable = secessionist_notification_cooldown_$rebellion_id$ }`。

**严格过滤检查**（基于德里陷落和干预属国内战模式）：
- **关键**：必须是排名最高的相同文化国家或防御方的 宿敌：`OR = { (culture = rebel_culture AND ordered_country by ranking_formula, max = 1) OR is_rival_of = defender }`（防止世界大战）
- 不得与防御方处于战争状态：`NOT = { is_at_war_with = defender }`（如德里陷落检查）
- **主要文化国家的停战协议例外：** 主要文化国家即使有停战协议也可以获取。为防止利用：**选项 A（推荐）：** 2.0x 成本乘数；**选项 B：** 50% 减少的惩罚（-25 稳定度，+0.5 厌战度，+12 敌意）；**选项 C：** 1.5x 成本 + 50% 减少的惩罚。理由：支持自己文化的独立是不对称战争；停战协议可能是被迫的；有意识的战略决策。检查：`OR = { culture = rebel_culture AND = { OR = { is_rival_of = defender is_leader_of_international_organization = { type = coalition target = defender } } NOT = { has_truce_with = defender } } }`。价格：`if = { limit = { culture = rebel_culture has_truce_with = defender } multiply = 2.0 }`
- 必须能够看到 局势：`can_see_situation = situation:secessionist_rebellion`（如宗教战争检查）
- 必须有外交容量：`diplomatic_capacity > 0`（防止过度扩张的国家）
- 不得处于内战中：`in_civil_war = no`（防止处理自己问题的国家）
- 不得破产：`is_during_bankruptcy = no`（如意大利战争检查）
- 必须有足够的 杜卡特：`gold >= (monthly_income_trade_and_tax * 2)`（如德里陷落 plea_neighbors 检查）
- 必须能够加入战争：`can_join_as_attacker = yes`（如干预属国内战检查）

## 机制设计（基于已验证的先例）

1. **早期警报和干预权获取（叛军达到 80-90%）：** 当叛军达到 80-90% 时创建局势。符合条件的国家收到通知（排名最高的相同文化国家或宿敌）。成本基于外交官成本、时间成本（20 间谍网强度，包括文化传统惩罚和反间谍）、距离、叛军规模。国家可以拒绝（可选，如 press_claims）。授予加入现有内战的权利，而非宣战。

2. **雇佣兵雇佣：** 获取的杜卡特被叛军用于在革命前雇佣雇佣兵（14+ 天集结）。给予叛军军事力量以在最初镇压中生存。

3. **潜在增强 - 秘密正规军训练：** 杜卡特资助秘密训练的正规军。**平衡解决方案：** 当叛军通过 `start_civil_war` 爆发时，它们形成叛军国家（`country_type = location`），受内置限制约束：人力池（人口/建筑）、招募能力（有招募建筑的地区）、训练速度（每个地区的团）、经济约束。自然平衡防止过度强大的军队。

4. **叛军爆发和内战（叛军达到 100%）：** 内战自动开始。叛军国家通过 `raise_levies` 获得部队。**关键机制：** 如果主要文化国家获取：叛军国家作为分离主义属国加入，该国成为 进攻方 战争领袖（防止 附庸 利用）。如果没有主要文化国家获取：叛军国家独立并领导战争（如 1821 年希腊独立战争）。已获取的国家可以使用 `join_war_as_attacker`（`reason = intervene`）加入。

5. **支持叛军集成：** 加快叛军进度。当叛军达到 80-90% 时，符合条件的国家可以立即获取。使支持叛军变得有价值。

6. **防御方反制策略：** 可以使用转换文化/宗教、增加容忍度、接受文化、镇压叛军。管理良好的防御方不太脆弱。有雇佣兵的叛军可以匹配防御方的初始力量。

7. **利用预防：** 通过消除强制参与防止利用 2（双重战争攻击）。防止利用 1（附庸使用宗主）——如果主要文化国家获取，它成为战争领袖（不是叛军国家/附庸）。早期警告系统（80-90%）给予准备时间。宗主特殊行动：取消附庸地位、创建不忠属国宣战理由、夺取领土、拒绝加入战争（重用现有机制）。

## 利用预防的额外修复

**修复 1：宗主战争领袖控制** - 宗主成为防御战争领袖（不是附庸），防止附庸在控制和平条款的同时利用宗主的军事力量。

**修复 2：宗主拒绝选项** - 从 `secessionists.txt` 中移除 `join_defensive_wars_always`。在 80-90% 时通知宗主，宗主可以选择加入或拒绝。

**修复 3：附庸叛乱检测** - 基于模式的怀疑分数（未核心领土、低稳定度、加速进度）。宗主可以取消附庸地位、创建不忠属国宣战理由、夺取领土、拒绝加入战争（重用现有机制）。

## 使用现有机制的实施

## 结论

干预权系统解决了强制参与、原所有者被忽视问题、缓慢的宣战理由创建、叛军军事弱点、通知垃圾邮件、世界大战升级和附庸利用。它使用已验证的机制（`press_claims`、`ordered_country`、`join_war_as_attacker`）并提供可选参与与平衡升级。原所有者优先级恢复（100 分奖励）。内置游戏限制提供自然平衡。成本考虑文化传统惩罚和反间谍。早期警报系统（80-90%）实现及时干预。资格限制防止世界大战，同时允许战略宿敌利用机会。

**术语说明**："干预权"授予加入现有内战（当叛军爆发时）的权利，而非宣战。
